# Calculator-python

A custom-built desktop calculator made with Python and Tkinter.

## ✨ Features

- Simple, clean GUI built with Tkinter
- Perform basic arithmetic operations (addition, subtraction, multiplication, division)
- Button-based interface for entering numbers and operators
- Lightweight — no external dependencies beyond Python's standard library

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Language | Python |
| GUI | Tkinter (Python's built-in GUI toolkit) |

## 📁 Project Structure

```
├── hi.py         # Main calculator application
└── README.md
```

## 🚀 Getting Started

### Prerequisites
- Python 3.x installed (Tkinter ships with most standard Python installations)

### Run Locally
```bash
python hi.py
```
